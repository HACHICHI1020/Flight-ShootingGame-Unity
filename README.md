FlightShootingGame (비행슈팅게임)

> 유니티 입문 1주일 차에 제작한 2D 비행 슈팅 게임 프로젝트입니다.
> 유니티의 기초적인 생명주기와 물리 충돌, 프리팹 시스템을 이해하기 위해 제작했습니다.

##  주요 기능
게임 제작에 있어 다음과 같은 주요 기능을 구현했습니다.

1.  **플레이어 이동 및 조작**
    * `Input.GeKEy'를 활용한 상하좌우 이동 구현
    * 화면 밖으로 나가지 않도록 `Mathf.Clamp`로 이동 제한
2.  **공격 시스템**
    * 스페이스바 입력 시 미사일 프리팹(Prefab) 생성 (`Instantiate`)
    * 미사일 발사 쿨타임(Cooltime) 적용
3.  **적(Enemy) 생성**
    * 랜덤한 위치에서 적 비행기 스폰
    * 플레이어를 향해 다가오거나 아래로 내려오는 기본적인 이동 로직
4.  **충돌 처리 및 점수**
    * `OnTriggerEnter`를 활용한 피격 판정
    * 적 격추 시 점수 UI 업데이트 및 폭발 이펙트 재생

##  기술적 고민
**문제 상황**
미사일을 계속 발사하니 Hierarchy 창에 PlayerRocket 객체가 무한히 쌓여 게임이 점점 느려지는 현상이 발생했습니다.

**해결 방법:**
* **시도 1:** 화면 밖으로 나가면 `Destroy()` 함수를 호출하여 객체를 파괴하도록 수정했습니다.
* **배운 점:** 생성(Instantiate)만큼이나 해제(Destroy)가 메모리 관리에 중요하다는 것을 배웠습니다.
